# Flourish Olaiya — Portfolio

Personal portfolio of Flourish Olaiya, AI & Machine Learning Engineer.

Live site: https://flourish-webpage.vercel.app

## Contents

- `index.html` — single-page portfolio (monochrome design, light/dark mode)
- `Flourish_Olaiya_CV.pdf` — current CV
- `profile.jpg`, `about.png`, `flourish_logo.png` — site assets

## Development

Static site with no build step. Open `index.html` directly or serve locally:

```bash
python -m http.server 8000
```
